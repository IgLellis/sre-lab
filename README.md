# SRE Lab - FastAPI + Docker

Projeto criado para estudo de práticas DevOps e SRE.

## 🚀 Tecnologias
- FastAPI
- Docker
- Docker Compose
- Grafana
- Prometheus

## 📦 Como rodar localmente

### Build
docker compose build

### Run
docker compose up

A aplicação sobe em:
http://localhost:8000
